# LinuxPowerCycle
A Linux tool to perform computer power cycling for hardware reliability tests.

1. OS Settings:
   a. Boot into Ubuntu as root
   b. Boot into Ubuntu without password

2. Install Python3 in Ubuntu.
3. Install pip 
4. Install TK
